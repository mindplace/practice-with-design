Enumerable

all?

takes block, returns boolean

any?

takes block, returns boolean

none?

returns boolean

find

returns first item to match block, else nil

grep(pattern)

returns array of objs for which pattern === obj

group_by

returns hash of groups grouped by block

partition

returns array with sub-arrays partitioned by block

reject

returns array without items that conform to block

drop_while

drops items that conform to block from front of array

max_by

returns greatest given block

min_by

returns smallest given block

sort_by

returns an array sorted by block requirement

count

takes block or arg, returns num

cycle(num)

executes block on obj num times

each_slice(num)

breaks into sets of num size, executes block per set

include?

returns boolean

inject

combines objs in array by applying specified operation

first(n)

returns n items from beginning of array
